# 🍽️ QR Menu – Digital Restaurant Ordering System

A responsive digital restaurant menu that allows customers to browse dishes, place orders, and track their order status in real time. The application also includes an owner dashboard for managing menu items and incoming orders using Firebase.

🌐 **Live Demo:** https://digitalmenuss.netlify.app

---

## 📖 About the Project

Traditional paper menus can be inconvenient to update and manage. This project was built to provide restaurants with a simple digital alternative where customers can access the menu, place orders, and receive live status updates directly from their phones.

On the restaurant side, owners can securely log in to manage the menu, monitor incoming orders, update order statuses, and control item availability in real time.

The application is built as a single-page web application using HTML, CSS, JavaScript, and Firebase.

## ✨ Features

### Customer Features

* Browse menu items by category
* Search dishes instantly
* Veg-only filter
* Add and remove items from the cart
* Automatic bill calculation
* Select table number before ordering
* Place orders digitally
* Track order status live

### Owner Features

* Secure login with Firebase Authentication
* Real-time order dashboard
* Update order status instantly
* Add new menu items
* Edit existing dishes
* Delete menu items
* Manage categories
* Toggle item availability
* Receive alerts for new orders

## 📱 How It Works

### Customer

1. Enter the table number.
2. Browse or search the menu.
3. Add dishes to the cart.
4. Place the order.
5. Track the order status until it is served.

### Restaurant Owner

1. Log in to the dashboard.
2. View new orders in real time.
3. Update the order status.
4. Manage menu items and categories.
5. Control dish availability instantly.
